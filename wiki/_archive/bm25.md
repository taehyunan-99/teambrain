## Summary
BM25(Best Match 25)는 TF-IDF 기반 확률적 정보 검색 모델로 키워드 검색의 사실상 표준이다. SQLite FTS5가 BM25를 기본 랭킹 함수로 내장하고 있어, [[sqlite-vec]]와 같은 DB에서 결합해 [[hybrid-search]]의 sparse retrieval 파트를 담당할 수 있다. **[손수정] 이 요약은 직접 편집한 내용이다.**

## Details
공식: `BM25(d,q) = Σ IDF(t) * (tf(t,d)*(k1+1)) / (tf(t,d) + k1*(1-b+b*|d|/avgdl))`
- k1=1.2~2.0, b=0.75 (기본 하이퍼파라미터)
- 문서 길이 정규화 포함

SQLite FTS5에서 BM25:
- `CREATE VIRTUAL TABLE notes_fts USING fts5(title, body)`
- BM25가 기본 랭킹 함수 (`rank` 컬럼, 음수값 → ORDER BY rank ASC)

[[hybrid-search]]에서의 역할:
- 정확한 용어 매칭(고유명사, 전문 용어)에 강점
- Dense([[sqlite-vec]])가 의미 유사도를 보완
- 두 결과를 [[rrf]]로 병합
