# slack #platform-announce 2024-09-17

**10:00 강민석**: [공지] 결제 API 배포 방식 변경 안내
**10:00 강민석**: 앞으로 결제 API는 빌드된 이미지를 기동하는 방식으로 배포됩니다. 서버에서 직접 git pull / 빌드하던 흐름은 제거됩니다
**10:01 강민석**: 머지 시 CI에서 build → test → 이미지 push까지 자동으로 돌고, 배포는 푸시된 이미지를 띄우는 식으로 전환됩니다
**10:02 강민석**: 롤백은 직전 정상 이미지 태그로 다시 기동하면 됩니다. 절차는 운영 문서에 정리해뒀어요
**10:03 강민석**: 정산/CS 쪽은 직접 영향 없지만 결제 배포 중에 잠깐 끊기는 구간이 보일 수 있어 참고차 공유합니다

**10:08 한지우**: 정산은 실시간 결제랑 분리돼 있어서 영향 없겠네요. 공유 감사합니다 🙏
**10:09 최민지**: 넵 배포 시간대 알려주시면 CS쪽 문의 들어와도 미리 대응할게요
**10:10 강민석**: 배포는 보통 트래픽 적은 시간대에 잡을 거라, 따로 잡히면 이 채널에 다시 공지할게요
